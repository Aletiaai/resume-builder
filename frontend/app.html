Configura tu API key de Gemini

CV Builder usa la IA de Google Gemini para generar tu currículum. Necesitas tu propia API key — es gratis y tarda menos de 2 minutos.

  1. Ve a Google AI Studio: aistudio.google.com
  2. Inicia sesión con tu cuenta de Google
  3. Haz clic en "Get API Key" → "Create API key"
  4. Copia tu API key y pégala aquí:

Tu API key se almacena de forma segura y nunca se comparte. La capa gratuita de Gemini es suficiente para generar varios currículums al día.
