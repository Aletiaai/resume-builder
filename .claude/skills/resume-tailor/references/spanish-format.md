# Spanish Resume Format

Use this file when building a resume in Spanish. All section titles, date formats, and language-specific rules apply.

---

## Spanish Section Titles

| Section | Spanish Title |
|---------|--------------|
| Summary | `RESUMEN` |
| Key Skills | `HABILIDADES CLAVE` |
| Relevant Work Experience | `EXPERIENCIA PROFESIONAL RELEVANTE` |
| Education | `EDUCACIÓN` |
| Languages | `IDIOMAS` |

> Note: The **Header** has no label in either language.

---

## Skills Section Format (Spanish)

In Spanish resumes, skills are organized in **labeled categories**, bold label followed by colon:

```
Agentes y Automatización: LangChain, LangGraph, OpenAI API, Claude API, Gemini API, FastAPI, REST APIs
Modelos e IA Generativa: Claude, Gemini, Whisper, Prompt Engineering (Chain of Thought, few-shot, one-shot)
Infraestructura Cloud: GCP (Cloud Run, Cloud Storage), AWS (Fargate, DynamoDB), Docker
Bases de Datos y Datos: ChromaDB, Vector Search, Embeddings (OpenAI, Google)
Plataformas Operativas: Zendesk, Twilio, WhatsApp API, Zapier, HubSpot, Mailerlite
Lenguajes: Python
Otros: Git, Visual Studio Code, WordPress
```

---

## Date Formats (Spanish)

| Element | Spanish Format |
|---------|---------------|
| Current job end date | `Actual` |
| Month abbreviations | Ene, Feb, Mar, Abr, May, Jun, Jul, Ago, Sep, Oct, Nov, Dic |
| Date range | `Ago 2020 – Actual` |

---

## Bullet Point Word Count (Spanish)

Spanish bullets are longer than English ones due to language structure:
- **Minimum:** 18 words
- **Maximum:** 25 words

All other bullet point rules from `section-rules.md` apply equally.

---

## Summary Rules (Spanish)

Same 3-part structure as English (see section-rules.md), but:
- Written naturally in Spanish — do not translate literally
- Maintain the formula: `[Adjetivo] [Título] con más de X años de experiencia [Propuesta de valor con métricas]`
- Case studies and extracurricular value follow in Spanish prose

---

## Education Section (Spanish)

Month abbreviations use Spanish (see table above):
```
Machine Learning Operations Fundamentals | Google | Sep 2025
Maestría en Diseño Mecánico | CICATA | Beca GE Aviation | Sep 2011 – Sep 2014
Ingeniería Mecatrónica | Tecnológico de Monterrey | Sep 2002 – May 2007
```

---

## Languages Section (Spanish)

Proficiency levels in Spanish:
| English | Spanish |
|---------|---------|
| Native | nativo/a |
| Fluent | fluido/a |
| Intermediate | intermedio/a |
| Basic | básico/a |

Format:
`Español (nativo) | Inglés (fluido, habla y escritura) | Francés (básico)`

---

## Portfolio / Highlighted Projects Section (Spanish only — optional)

If the user's resume includes a portfolio or highlighted freelance project, add a section:

**Label:** `PORTAFOLIO | PROYECTO DESTACADO`

Format:
- Project name in bold, followed by GitHub or URL link
- 2–4 bullet points following the same bullet rules
- Only include if the project is directly relevant to the target job description

---

## Key Differences vs English Resume

| Element | English | Spanish |
|---------|---------|---------|
| Section: Experience | Relevant Work Experience | Experiencia Profesional Relevante |
| Section: Skills | Key Skills | Habilidades Clave |
| Skills format | Flat pipe-separated list | Labeled categories |
| Bullet length | 12–20 words | 18–25 words |
| Current date label | Current | Actual |
| Portfolio section | Not included | Include if relevant |
| Language proficiency | Fluent, Native, Intermediate | fluido, nativo, intermedio |
